你所不知道到的神奇寶貝

簡介:

靈寶可夢特別篇（台譯︰神奇寶貝特別篇，港譯︰寵物小精靈特別篇，日文︰ポケットモンスター☆SPECIAL Pocket Monsters ☆ SPECIAL，英文︰Pokémon Adventures）是由日下秀憲作腳本，真斗和山本智作畫的漫畫系列，簡寫為ポケスペ/ポケSP（PokéSpe）。 和動畫不同，特別篇漫畫的設定更加基於遊戲，但是又有明顯的不同。特別篇的21個主角的名字正是來自於主要的21款遊戲（不含重製），而他們冒險的地方也是以這些遊戲的地區為主。與動畫花大量篇幅闡述傳說的寶可夢難以被捕捉的情況不同，特別篇會出現捕捉傳說的寶可夢甚至使用它們的情況，這點來說和遊戲是非常相似的。同時漫畫中偶爾也會出現角色或寶可夢死亡這樣的消極情況，但總體來說特別篇的格調還是相當積極的。 特別篇漫畫曾於小學年生誌、精靈寶可夢仙境和精靈寶可夢愛好者上連載，均已停止連載。目前在Corocoro一番!上連載，內容為太陽·月亮篇。另外終極紅寶石·原始藍寶石篇還在Club Sunday（クラブサンデー）以及Sunday Web Every（サンデーうぇぶり）網上連載，而一時間中止連載的黑2·白2篇也緊接著終極紅寶石·原始藍寶石篇之後在Sunday Web Every上恢復連載。 精靈寶可夢的創始人田尻智說：「這個漫畫和我想要表達的世界是最為接近的[1]」，精靈寶可夢系列插畫師杉森建亦推薦過此作品。 遊戲開發者石原恆和也高度評價這個系列，他說：「我希望所有的精靈寶可夢愛好者都來看這個系列[2]。」
